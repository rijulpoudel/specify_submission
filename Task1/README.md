# Task 1 - DashboardButton

A simple React + TypeScript button component for the Specify Assistant Student Developer assessment.

## What it does

Renders a button that logs `Dashboard button clicked` to the console when clicked.

## Notes

- Followed Specify7 conventions from the repo
- Used named export with `export function`
- Props use `type` with `readonly`
- Used `React.useCallback` for the click handler to match Specify7's namespaced style

## Assumptions

- No styling was added since the task focuses on component structure and behavior, not visual design.
