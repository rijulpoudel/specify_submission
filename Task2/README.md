# Task 2 - group_by_category

A Python function for the Specify Assistant Student Developer assessment.

## What it does

Takes a list of items and groups them by category.

## Notes

- Return type is `dict` as specified, but internally typed as `dict[str, list[str]]` for clarity
- Empty list returns `{}` with no errors
- Each item is expected to have both `name` and `category` keys
